So how can you use these tools?

  • Think of a story that branches, or where one discovery leads to another.
  • Create HTML pages that link to reveal each part of the story.
  • Incorporate images and audio
  • Make a CSS style sheet to get a consistent, appealing look.
  • Add JS to create animations and other effects.


Here's a suggested work process:

  • Write your files in a text editor (I'm using Atom)
  • Create a folder to put all your files in (you can make subfolders for images and other media, if desired)
  • Open the files in Firefox (which has excellent developer tools)
  • Open a different browser so you can look things up-- you'll do that a lot! (again, W3Schools is helpful, as is Mozilla Developer Network)
  • To share your finished product, create a repository at GitHub and turn on GitHub pages


    • One final thought...even if you use other web-based tools to create content, you'll likely find that having some knowledge about the underlying tools discussed here will help you and give you insight into how those tools work

    Have fun coding!
