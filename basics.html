HTML basics

HTML (hypertext markup language) uses tags (contained within angle brackets) to let the browser know what to display (and how). You probably noticed several <p> tags in the source code for the previous page (index.html), for example. These indicate the start of a new paragraph. The end of the paragraph is indicated by the closing </p> tag.
The basic structure of any page is a "head" section, where you'll put the title of the page and sometimes load other files (more on that later). What the viewer actually sees on the page is contained in the "body."


Here's a template for a basic page:


<html>
<head>
<title>
Name to display in the browser tab
</title>
</head>
<body>
Content of the page.
</body>
</html>


Within the body, Use <h1> for headings (h2, h3, etc are smaller subheadings) and <p> for normal (paragraph) text. Note that a tag is "closed" using a forward slash. This lets the browser know when, for example, a paragraph is complete. One exception to this is the <br> tag, because it does not enclose any content-- it simply produces a line break. Tags can be nested inside of other tags. For example, there are many tags opened and closed within the body, which itself is within the HTML
