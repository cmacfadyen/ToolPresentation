There are many helpful tools available for designing learner experiences, however you may want to consider building something "from scratch."

Biggest pro: you can design the activity exactly as you want it
Biggest con: it can be a lot of work


What you're looking at is a simple web page, coded using only HTML. The process is:

  • Open a text editor (such as Atom, Sublime, Notepad++)
  • Create a text file with HTML "tags" (more on that in a bit)
  • Save the file with the extension .html or .htm
  • To view the file, use Ctrl-O or Cmd-O to open the file in any browser (note that you'll need to refresh each time you save new changes to the file)

Take a look at the actual code for this page by choosing "View Source" in your browser (typically, Ctl-U on a PC or Cmd-U on Mac, or via the menu).
